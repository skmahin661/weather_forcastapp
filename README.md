# 🌤 Weather Forecast AI

A Deep Learning-based Weather Forecasting web application built with Flask and TensorFlow.

## Features

- ☀️ Sunny / 🌧 Rainy Prediction
- Deep Learning Model (.keras)
- Google Weather Inspired UI
- Confidence Score
- Responsive Design
- Glassmorphism Interface

## Tech Stack

- Python
- Flask
- TensorFlow
- HTML
- CSS
- JavaScript

## Installation

```bash
pip install -r requirements.txt
```

Run:

```bash
python app.py
```

Open your browser:

```
http://127.0.0.1:5000
```

## Model

Uses:

- best_model.keras
- scaler.pkl
